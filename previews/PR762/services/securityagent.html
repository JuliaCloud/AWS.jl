Main.Securityagent.create_agent_space — Function
create_agent_space(name)
create_agent_space(name, params::Dict{String,<:Any})

Creates an agent space record

Arguments

  • name: Name of the agent space

Optional Parameters

Optional parameters can be passed as a params::Dict{String,<:Any}. Valid keys are:

  • "awsResources": AWS resource configurations associated with the agent space
  • "codeReviewSettings": Configuration for code review analysis, including controls scanning and general purpose scanning settings
  • "description": Description of the agent space
  • "kmsKeyId": Identifier of the KMS key used to encrypt data. Can be a key ID, key ARN, alias name, or alias ARN. If not specified, an AWS managed key is used.
  • "tags": Tags to associate with the agent space
  • "targetDomainIds": Target domain IDs to associate with the agent space
