Updates the connection preferences for a partner account, modifying access settings and exclusions.

Arguments

  • access_type: The access type setting for connections (e.g., open, restricted, invitation-only).
  • catalog: The catalog identifier for the partner account.
  • revision: The revision number of the connection preferences for optimistic locking.

Optional Parameters

Optional parameters can be passed as a params::Dict{String,<:Any}. Valid keys are:

  • "ExcludedParticipantIdentifiers": The updated list of participant identifiers to exclude from connections.
